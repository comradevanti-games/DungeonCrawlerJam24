# Scene live-cycle

## Game

1. Awake
2. Pre-game
   1. Prepare start
   2. Prepare done
3. Game
   1. Game start
   2. Round loop
      1. Round start - New round begins
      2. Action start - All actors have chosen action
      3. Action complete - All actors have completed their action
      4. Round done - The round is over
   3. Game done
